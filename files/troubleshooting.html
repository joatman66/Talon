üö® Common Issues & Solutions

Dashboard shows "Loading..." forever
1. Open browser console (F12 ‚Üí Console tab)
2. Look for errors (usually red text)
3. Check if dashboard_proxy.php is accessible
4. Test: http://yourserver.com/dashboard_proxy.php directly
"No response received" in test_api.php
1. Verify API URL is correct
2. Check firewall - server must make outbound HTTPS connections
3. Verify credentials (customer code, access code, username)
4. Check server error logs
Aircraft show but no operations data
1. Verify LOCATION name matches Talon exactly (case-sensitive)
2. Confirm there are scheduled flights for today
3. Check activitytype filter (currently set to 'Flight')
4. Test operations endpoint in test_api.php
"XML Parsing Failed"
1. API is returning an error message instead of XML
2. Check raw response in test_api.php
3. Verify all credentials are correct
4. Contact Talon support if error persists
