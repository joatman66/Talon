PROBLEM: Dashboard stuck on "Loading..."
SOLUTION: 
  1. Press F12 in browser â†’ Check Console for errors
  2. Test proxy directly: http://yourserver.com/dashboard_proxy.php
     (Should return JSON data)
  3. Verify credentials in dashboard_proxy.php

PROBLEM: "No response received" in test_api.php
SOLUTION:
  1. Verify API_URL is correct
  2. Check firewall settings (server needs outbound HTTPS)
  3. Confirm credentials are correct
  4. Check server error logs

PROBLEM: Aircraft show but no operations
SOLUTION:
  1. Verify LOCATION matches Talon exactly (case-sensitive!)
  2. Check if you have flights scheduled TODAY
  3. Try different date range in dashboard_proxy.php

PROBLEM: "XML Parsing Failed"
SOLUTION:
  1. API returning error instead of XML
  2. Check raw response in test_api.php
  3. Verify ALL credentials
  4. Contact Talon support
